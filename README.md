# SendingEmailSH
Envio de Emails para GMAIL|OUTLOOK por um shell script


![SendEmail](https://github.com/Gabryel8818/SendingEmailSH/blob/master/sendmail.gif)

## Como usar

``` bash
$ chmod +x client-email.sh
$ ./cliente-email.sh

```
